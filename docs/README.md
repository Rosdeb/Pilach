## 🎨 Theme & Typography

Pilach Message App is built on a design system prioritizing dynamic readability:
- **Light Theme Background:** Slate/grey tones (`#F8FAFC`, `#F1F5F9`) ensuring high readability and clean lines.
- **Dark Theme Background:** Deep slate and true-black tones (`#0F172A`, `#000000`) for low-light comfort.
- **Typography:** Modern sans-serif text hierarchy (via Inter/Roboto styles) utilizing varying font weights to denote information priority.
